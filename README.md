## 📁 Repository Structure

```bash
├── assets/             # Theme fonts, images, and scripts
├── config/             # Theme settings and schema
├── layout/             # Master layout files (theme.liquid)
├── locales/            # Multi-language support files
├── sections/           # Reusable UI components
├── snippets/           # Reusable code fragments
└── templates/          # JSON templates for all pages
```

## 🛫 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/syaheka/shopify-premium-theme.git
   ```
2. **Install Shopify CLI** (if not already installed):
   ```bash
   npm install -g @shopify/cli @shopify/theme
   ```
3. **Connect to your store**:
   ```bash
   shopify theme dev
   ```

---

## 🌟 Customization

This theme is built to be extended. You can modify the global styles in `assets/base.css` and configure global settings in the Shopify Theme Editor under **Theme Settings**.
